// 03 — Featured Project

Built something
that solved a real problem.

Capstone Project — Lead Developer

LAN-Based File Transfer &
Remote Management System

↳ Full-stack desktop application for laboratory PC management

Designed and built a centralized remote management platform for computer laboratory environments. The system enables a single administrator to monitor, control, and distribute files to multiple student workstations simultaneously — replacing the need for manual, machine-by-machine management. Built entirely with custom TCP socket communication over a local area network, the application delivers real-time responsiveness without relying on third-party remote access tools.

Real-time screen monitoring — live view of student PC activity from the admin workstation
Remote PC control — lock/unlock, restart, and screen blackout for individual or all clients
Bulk file distribution — transfer files from the admin PC to multiple client machines simultaneously
Custom TCP protocol — lightweight client-server architecture built without third-party networking libraries
WPF admin dashboard — clean, organized interface for managing all connected lab machines in one view
Performance optimized — stress-tested and debugged for reliability in a live multi-PC lab environment
C# WPF / XAML .NET Framework TCP Sockets LAN Networking Multi-threading Custom Protocol
